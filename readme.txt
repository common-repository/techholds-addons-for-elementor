== Description ==
**TechHolds Addons For Elementor** is a Easy to use plugin by TechHolds that adds widgets like 5-star rating for your posts/pages/products.

**Widgets List:**

- [**Post/Page/product Rating**](https://techholds.com/elementor-addons/)

== Installation ==
1. Upload the `techholds-addons-for-elementor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Fontawesome font not loading on forntend? =

Enable fonts support. Go to Elementor -> Settings -> Advance -> Load Font Awesome 4 Support and change to Yes.

= Can I customizable fonts color, size of icons? =

Yes. All controls are fully customizable.
